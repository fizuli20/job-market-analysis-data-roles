## Project Overview

This portfolio analyzes the **`lukebarousse/data_jobs`** dataset, which contains real-world job postings for data roles (Data Analyst, Data Engineer, Data Scientist) across multiple countries. The analysis reveals:

- Salary distributions by job title and country
- Top in-demand skills per role
- Quarterly and monthly hiring trends
- Geographic job market variations

## Notebook Breakdown

## Installation & Setup

### Prerequisites
- Python 3.8+
- Jupyter Lab or VS Code with Jupyter extension

### Quick Start

1. Clone and navigate to the directory:
```bash
cd job-market-analysis
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/Scripts/activate  # Windows
# or: source venv/bin/activate  # macOS/Linux
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Launch Jupyter:
```bash
jupyter lab
```

5. Open any notebook and run cells to reproduce the analysis.
